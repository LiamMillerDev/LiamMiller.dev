- Showcasing web development projects

## Technology Stack

- CSS
- JavaScript

## Setup and Usage
To view this portfolio:
1. Clone the repository: `git clone https://github.com/yourusername/LiamMiller.dev.git`
2. Open the `index.html` file in a browser.
